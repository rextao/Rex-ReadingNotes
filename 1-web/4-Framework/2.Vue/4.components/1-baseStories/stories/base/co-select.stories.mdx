### select的简洁用法

如`options`返回字段非label与value，可以通过配置`label`和`val`解决

<Canvas>

                { label: '深圳', value: 4 },
            ],
        }}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="options使用非label与value"
        args={{

                { text: 'options字段为label与id', id: 4 },

    </Story>
</Canvas>
