### 隐藏单选时的radio按钮
在设置：props:{ checkStrictly: true }时，产品有时需要将前面的 ⭕ 隐藏掉，且点整行可以触发，可以设置： `hideRadio = true`

`hideRadio = true` 默认在8个字符长度，以上会显示tooltip

<Story
    name="隐藏单选时的radio按钮"
    args={{
        hideRadio: true,
        props:{ checkStrictly: true }
    }}
>
    {Template.bind({})}
</Story>

```html
<co-cascader
    :hideRadio="true" ,
    :props="{ checkStrictly: true }"
></co-cascader>
```

### 去除最后的无数据面板
一般后端返回的树状数据是递归的，children最后无数据会返回空数组，cascader会渲染一个空数据看板

    :setNull="true" ,
