### ckeckbox的简洁用法

如`options`返回字段非label与value，可以通过配置`label`和`val`解决

<Canvas>
    <Story
        name="checkbox的简洁用法"
        args={{
            options: [
                { label: '上海', value: 1 },
                { label: '北京', value: 2 },
                { label: '广州', value: 3 },
                { label: '深圳', value: 4 },
            ],
        }}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="options使用非label与value"
        args={{
            val: 'id',
            label: 'text',
            options: [
                { text: '上海', id: 1 },
                { text: '北京', id: 2 },
                { text: '广州', id: 3 },
                { text: 'options字段为label与id', id: 4 },
            ],
        }}
    >
        {Template.bind({})}
    </Story>
</Canvas>
